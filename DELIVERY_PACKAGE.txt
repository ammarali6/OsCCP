USAGE WORKFLOW
═════════════════════════════════════════════════════════════════════════════

Basic Workflow:

1. Launch: python main.py
   → GUI window opens

2. Add Processes:
   → Enter Arrival Time (e.g., 0)
   → Enter Burst Time (e.g., 5)
   → Enter Priority (e.g., 1)
   → Click "Add Process"
   → Repeat for more processes

3. Select Algorithm:
   → Choose one of 5 options
   → Set quantum if needed (for Round Robin/MLFQ)

4. Run Simulation:
   → Click "▶ Start" button
   → System executes algorithm and displays results

5. View Results:
   → Click "📊 Metrics" tab for performance statistics
   → Click "📈 Gantt Chart" tab for visual timeline
   → Click "📋 Details" tab for detailed process info

6. Get Recommendations (Optional):
   → Click "Get Recommendations"
   → Read AI-based algorithm suggestions

7. Export (Optional):
   → Click "Save Chart" to export Gantt as PNG
   → Save to desired location
