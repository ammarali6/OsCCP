💻 WORKFLOW GUIDE
═════════════════════════════════════════════════════════════════════════════

Complete User Workflow:

1. Start Application
   $ python main.py

2. Add Processes
   • Enter Arrival Time, Burst Time, Priority
   • Click "Add Process"
   • Repeat for more processes

3. Select Algorithm
   • Choose from: FCFS, SJF, Priority, Round Robin, MLFQ
   • Set quantum if needed

4. Run Simulation
   • Click "▶ Start"

5. View Results
   • Metrics tab: Performance statistics
   • Gantt Chart tab: Visual timeline
   • Details tab: Process information

6. Get Recommendations (Optional)
   • Click "Get Recommendations"
   • See AI suggestions

7. Export (Optional)
   • Save Gantt chart as PNG

📊 ALGORITHMS AT A GLANCE
═════════════════════════════════════════════════════════════════════════════

Algorithm       File                Best For              Complexity
─────────────────────────────────────────────────────────────────────
FCFS            algorithms/fcfs.py       Batch jobs              O(n log n)
SJF             algorithms/sjf.py        Short jobs              O(n²)
Priority        algorithms/priority.py   Real-time systems       O(n²)
Round Robin     algorithms/round_robin.py  Time-sharing          O(n × q)
MLFQ            algorithms/mlfq.py       Mixed workloads         O(n × levels)
