TROUBLESHOOTING
═══════════════

Problem 1: Python not found
──────────────────────────
Error: 'python' is not recognized as an internal or external command
Solution: 
1. Reinstall Python and check "Add Python to PATH"
2. Restart Command Prompt after Python installation
3. Try: python3 instead of python

Problem 2: ModuleNotFoundError: No module named 'matplotlib'
────────────────────────────────────────────────────────────
Error: ModuleNotFoundError or ImportError
Solution:
1. Ensure Python 3.7+ is used
2. Run: pip install --upgrade pip
3. Run: pip install matplotlib numpy
4. If using virtual environment, activate it first

Problem 3: Permission denied on Linux/macOS
─────────────────────────────────────────────
Error: Permission denied when running python
Solution:
1. Add execute permission: chmod +x quickstart.py
2. Run with python: python quickstart.py
3. Or use: python3 quickstart.py

Problem 4: GUI doesn't launch
──────────────────────────────
Error: Window doesn't appear or crashes
Solution:
1. Ensure Tkinter is installed (usually comes with Python)
2. On Linux, may need: sudo apt-get install python3-tk
3. Check for error messages in terminal
4. Try test_scenarios.py first

Problem 5: Chart visualization not working
──────────────────────────────────────────
Error: Gantt chart doesn't display
Solution:
1. Upgrade matplotlib: pip install --upgrade matplotlib
2. Check if display is available (on remote connections)
3. Save chart instead: Use "Save Chart" button
4. Verify numpy is installed: pip install numpy


VIRTUAL ENVIRONMENT (Optional but Recommended)
═══════════════════════════════════════════════

Using Virtual Environment isolates your project:

Windows:
────────
1. Create venv: python -m venv scheduler_env
2. Activate: scheduler_env\Scripts\activate
3. Install: pip install -r requirements.txt
4. Run: python main.py
5. Deactivate: deactivate

macOS/Linux:
────────────
1. Create venv: python3 -m venv scheduler_env
2. Activate: source scheduler_env/bin/activate
3. Install: pip install -r requirements.txt
4. Run: python main.py
5. Deactivate: deactivate
