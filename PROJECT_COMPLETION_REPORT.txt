✅ Round Robin
   └─ algorithms/round_robin.py
   └─ Preemptive with configurable quantum

✅ MLFQ (Multi-Level Feedback Queue)
   └─ algorithms/mlfq.py
   └─ Adaptive scheduling with 3 queues

HOW TO USE
════════════════════════════════════════════════════════════════════════════

1. Installation
   $ pip install -r requirements.txt

2. Run Application
   $ python main.py

3. Add Processes
   • Fill in Arrival Time
   • Fill in Burst Time
   • Fill in Priority
   • Click "Add Process"

4. Select Algorithm
   • Choose from: FCFS, SJF, Priority, Round Robin, MLFQ
   • Set quantum if using Round Robin

5. Run Simulation
   • Click "▶ Start" button
   • Results appear immediately

6. View Results
   • Metrics tab: Performance statistics
   • Gantt Chart tab: Visual timeline
   • Details tab: Process information

7. Get Recommendations
   • Click "Get Recommendations"
   • See AI suggestions

8. Export Results
   • Use "Save Chart" to export as PNG
